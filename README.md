#### logstash-test

Basic little program to send log data over to Logstash. It randomly selects a
[Zippy the Pinhead](http://www.zippythepinhead.com/) quote and logs it.

Simple.

```
Usage of ./logstash-test:
  -logstash string
    	logstash server
  -sleep int
    	time to sleep in seconds (default 5)
```
